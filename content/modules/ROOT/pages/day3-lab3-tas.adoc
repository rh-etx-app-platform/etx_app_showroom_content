== Overview

Trusted Artifact Signer (TAS) provides keyless signing for container images and other software artifacts using Sigstore technologies: cosign, Fulcio, and Rekor.

== Exercise: Sign a Container Image

Sign the container image built in Lab 1 using cosign and TAS:

[source,bash]
----
cosign sign --fulcio-url=<fulcio-url> --rekor-url=<rekor-url> <image-reference>
----

== Exercise: Verify a Signed Image

Verify the signature of a signed container image:

[source,bash]
----
cosign verify --rekor-url=<rekor-url> <image-reference>
----
