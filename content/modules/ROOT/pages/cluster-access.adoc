== Accessing ConfigMap Data

The provision data is stored in a ConfigMap labeled with `demo.redhat.com/infra: "true"` for automated discovery by RHDP components.

To view all provision data:

[source,bash,role=execute]
----
oc get configmap etx-lab-provision-data -n openshift-gitops -o yaml
----

To extract a specific value:

[source,bash,role=execute,subs="attributes"]
----
# Get Keycloak URL
oc get configmap etx-lab-provision-data -n openshift-gitops \
  -o jsonpath='{.data.keycloak_url}'

# Get ArgoCD URL
oc get configmap etx-lab-provision-data -n openshift-gitops \
  -o jsonpath='{.data.argocd_url}'
----
