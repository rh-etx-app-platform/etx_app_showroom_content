== Install the Dev Spaces Operator

=== Install via OperatorHub

. Open {openshift_console_url}[^] with an admin user.
. In the OpenShift Web Console, go to menu:Ecosystem[Software Catalog]
. Search for *Red Hat OpenShift Dev Spaces* and click btn:[Install]
. Accept the defaults (All namespaces, Automatic approval) and click btn:[Install]
. Wait for the operator to reach the `Succeeded` phase

image::devspaces-operator-installed.png[Dev Spaces Operator installed]

Verify from the CLI:

[source,bash]
----
oc get csv -n openshift-operators | grep devspaces
----

A similar output should be displayed:

[bash]
----
devspacesoperator.v3.27.1           Red Hat OpenShift Dev Spaces   3.27.1    devspacesoperator.v3.27.0           Succeeded
----

The DevWorkspace Operator is installed automatically as a dependency.

image::devspaces-operator-list.png[Dev Spaces Operators Installed]
