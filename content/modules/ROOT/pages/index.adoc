= ETX App Platform Workshop
:navtitle: Workshop Introduction

Welcome to the Emerging Tech Experience (ETX) App Platform Workshop -- a three-day, hands-on enablement that will guide you to building a modern and secure platform on OpenShift using Red Hat technologies.

== What You Will Build

Over three days you will stand up the key services of a modern application platform on OpenShift, wire them together, and secure the entire software delivery process:

* A *cloud-native development environment* with Red Hat Dev Spaces, devfiles, and AI-assisted coding.
* A *complete CI/CD pipeline* using OpenShift Pipelines (Tekton) and OpenShift GitOps (Argo CD) with multi-environment promotion.
* An *Internal Developer Portal* powered by Red Hat Developer Hub, with Golden Path Templates for self-service project creation.
* A *Trusted Software Supply Chain* using Trusted Artifact Signer (TAS) and Trusted Profile Analyzer (TPA) for artifact signing, attestation, and vulnerability analysis.

== Workshop Structure

[cols="1,3",options="header"]
|===
| Day | Focus

| *Day 1*
| *Lab Setup & Git Foundations* -- Access your OpenShift cluster, configure your environment, and establish branching strategies for collaborative development.

| *Day 2*
a| *Software Factory & Application Lifecycle* -- Create a complete Software Factory to develop, build, deploy, and monitor using Red Hat solutions.

* Create a Quarkus application with an integrated setup of GitLab, OpenShift Dev Spaces, OpenShift Pipelines(Tekton), OpenShift GitOps (Argo CD), and OpenTelemetry.
* Provision AMQ Streams (Kafka), and PostgreSQL, as supporting services for applications.
* Complete the modernization journey of a legacy application with Migration Toolkit for Applications (MTA).
* Integrate services on an internal developer portal (IDP) based on Red Hat Developer Hub.

| *Day 3*
| *Trusted Software Supply Chain* -- Extend your pipelines adding the Red Hat Trusted Software Supply Chain (TSSC) components; Red Hat Trusted Artifact Signer (TAS), and Red Hat Trusted Profile Analyzer (TAP) to enable the integration of cryptographic signing and trust verification.
|===
