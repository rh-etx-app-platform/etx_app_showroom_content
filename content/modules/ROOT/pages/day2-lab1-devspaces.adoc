= Lab 1: Dev Spaces and IDE Configuration
:navtitle: Dev Spaces & IDE Setup

== Overview

In this section you will provision your lab environment, deploy Red Hat OpenShift Dev Spaces, and configure your development tools including Git integration and the AI Code Assistant.

== Provision the Lab Environment

If you have not already done so on Day 1, order a https://catalog.demo.redhat.com/catalog/babylon-catalog-prod/order/published.ocp4-cluster.prod[Red Hat OpenShift Container Platform Cluster (Multi-Cloud)] from the Red Hat Demo Platform with the following options:

[cols="1,2",options="header"]
|===
| Option | Value

| Activity
| Practice / Enablement

| Purpose
| Conduct internal training/enablement

| Cloud Provider
| cnv

| OpenShift Version
| 4.20

| Cluster Size
| multinode

| OpenShift Worker Count
| 3

| Create Users on Cluster
| 5

| Enable OpenShift Lightspeed
| Disabled
|===

Once your cluster is provisioned, log in and verify access:

[source,bash,subs="attributes"]
----
oc login --server=<your-api-url>
oc whoami
oc get nodes
----

== Create a Dev File

Create a `devfile.yaml` to define your workspace configuration, including:

* Base container image
* IDE plugins and extensions
* Git repository sources
* Resource limits

[source,yaml]
----
schemaVersion: 2.2.0
metadata:
  name: my-workspace
components:
  - name: tools
    container:
      image: registry.redhat.io/devspaces/udi-rhel8
      memoryLimit: 4Gi
----
