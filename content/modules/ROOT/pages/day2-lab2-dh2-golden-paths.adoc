== Overview

Red Hat Developer Hub provides a self-service portal for developers, built on Backstage.
In this section you will create a Golden Path template, integrate with Dev Spaces, and configure authentication.

== Create a Golden Path Template

A Golden Path defines a standardized, opinionated way to create new projects that follow organizational best practices.

Your Golden Path template will include:

* Pre-configured CI pipeline (Tekton)
* GitOps repository structure for Argo CD
* Dev Spaces devfile for consistent development environments
* Observability configuration (OpenTelemetry)
