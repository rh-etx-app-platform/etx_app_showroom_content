== Exercise 1: Clone and Explore

=== Step 1: Access GitLab

Open your browser and navigate to GitLab:

[source,bash,role=execute]
----
echo "GitLab URL: {gitlab_url}"
----

Login with your credentials:

* Username: `{user}`
* Password: `{password}`

=== Step 2: Clone the Repository

In the terminal, clone the `parasol-insurance-lab` repository:

[source,bash,role=execute]
----
cd ~
git clone {gitlab_url}/software-factory/parasol-insurance-lab.git
cd parasol-insurance-lab
----

.Expected Output
[source,text]
----
Cloning into 'parasol-insurance-lab'...
remote: Enumerating objects: 150, done.
remote: Counting objects: 100% (150/150), done.
remote: Compressing objects: 100% (95/95), done.
remote: Total 150 (delta 48), reused 150 (delta 48)
Receiving objects: 100% (150/150), 45.23 KiB | 1.51 MiB/s, done.
Resolving deltas: 100% (48/48), done.
----

=== Step 3: Explore the Repository

View the repository structure and current branch:

[source,bash,role=execute]
----
ls -la
git status
git branch
----

.What You Should See
* The repository files and directories
* Current branch: `main`
* Working directory clean (no uncommitted changes)

[TIP]
====
The `git status` command is your best friend. Use it frequently to understand the current state of your repository.
====
