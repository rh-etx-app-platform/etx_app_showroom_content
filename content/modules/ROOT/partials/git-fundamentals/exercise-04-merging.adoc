== Exercise 4: Merging Branches

=== Step 1: Merge First Feature

Switch to main branch:

[source,bash,role=execute]
----
git checkout main
----

Merge the README update:

[source,bash,role=execute]
----
git merge feature/update-readme
----

.Expected Output
[source,text]
----
Updating c9d2a45..b7e4c21
Fast-forward
 README.md | 3 +++
 1 file changed, 3 insertions(+)
----

[NOTE]
====
*Fast-forward merge:* Git simply moves the `main` pointer forward because there were no other changes on `main`. This is the simplest type of merge.
====

=== Step 2: Merge Second Feature

Merge the version update:

[source,bash,role=execute]
----
git merge feature/update-version
----

This creates a merge commit because both branches modified the repository.

=== Step 3: Clean Up Feature Branches

After merging, delete the feature branches:

[source,bash,role=execute]
----
git branch -d feature/update-readme
git branch -d feature/update-version
----

Verify only main remains:

[source,bash,role=execute]
----
git branch
----
